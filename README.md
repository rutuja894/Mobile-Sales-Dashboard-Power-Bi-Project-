# Mobile Sales Dashboard(Power Bi Project)
This Mobile Sales Dashboard provides a comprehensive overview of sales performance across cities, brands, and models. It highlights total sales (769M), quantity sold (19K), transactions (4K), and average price. The dashboard analyzes monthly trends, payment methods, customer ratings, and day-wise sales to support data-driven business decisions.
